Serviço RESTful em Spring Boot para gerenciar pedidos. O objetivo é praticar
mapeamento JPA com tipos de data/hora e operações CRUD.

Endpoints RESTful:
POST /pedidos — criar novo pedido |
GET /pedidos — listar todos os pedidos |
GET /pedidos/{id} — buscar pedido por ID |
DELETE /pedidos/{id} — deletar pedido por ID
